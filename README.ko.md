## PercentSnap이란?

**PercentSnap**은 일상에서 실제로 계산해야 하는 것들에 집중한 빠른 퍼센트 계산기입니다. 매장 할인, 세일가, 가격 책정용 마크업과 마진, 부가세, 퍼센트 변화, 원래 값을 찾는 역퍼센트, 퍼센트 포인트, 팁 분배, 복리, 어떤 수가 전체에서 차지하는 비율.

군더더기 없이 빠른 답. 광고 없음, 가입 없음, 추적 없음 — 이력, 고정, 설정은 기기에 남습니다.

## 주요 기능

### 계산기
- **할인** — 세일가와 절약 금액
- **퍼센트 변화** — 증가 또는 감소
- **역퍼센트** — 퍼센트 적용 전의 값
- **원래 가격** — 할인 전
- **마크업** — 원가와 마크업에서 판매가
- **마진 / 이익**
- **부가세 / 세금** — 더하기 또는 빼기
- **퍼센트 포인트**
- **숫자의 %**
- **무엇이 몇 %**
- **필요 퍼센트 변화**
- **팁과 분배**
- **복리**
- **전체에 대한 비율**

### 이력과 편의
- **로컬 이력**
- **자주 쓰는 계산 고정**
- **한 번 탭으로 복사**
- **이력 공유 / 내보내기**
- **형식·반올림** 조정 가능

### 개인정보 보호
- **광고 없음**
- **추적 없음**
- **계정 없음**
- **기기 내**

## 사용 시나리오

| 상황 | PercentSnap이 하는 일 |
|------|--------------------------|
| 매장 세일 | 할인 → 정가와 % → 세일가와 절약 |
| 식당 팁 | 팁 + 분배 → 계산서, %, 인원 → 1인당 |
| 부가세 역산 | 부가세 → 세금 포함 → 세금 제외 |
| 재판매 가격 | 마크업 → 원가 + 마크업 → 판매가 |
| 제품 이익 | 마진 → 원가 + 판매가 |
| "12% 오르면 얼마" | 퍼센트 변화 |
| "₩240,000이 -20% 후, 원래는?" | 역퍼센트 |
| 전년 대비 변화 | 필요 퍼센트 변화 |
| 장기 저축 | 복리 |
| 설문 조사 | 전체 비율 |

## 작동 방식

**기본 계산기 대신 전용 앱이 왜 필요한가요?**
기본 계산기에는 "할인", "부가세 빼기", "역퍼센트", "팁 + 분배" 전용 UI가 없습니다. PercentSnap은 각 입력에 라벨을 붙이고 사용한 공식을 보여줍니다.

**계산이 온라인에서 이루어지나요?**
아닙니다. 모두 기기 내에서, 인터넷 없이.

**자주 쓰는 계산을 저장할 수 있나요?**
네, 고정하면 됩니다.

**오프라인 작동?**
완전합니다.
