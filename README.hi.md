## PercentSnap क्या है?

**PercentSnap** एक केंद्रित, तेज़ प्रतिशत कैलकुलेटर है उन हिसाबों के लिए जो रोज़ ज़रूरी होते हैं: दुकान में डिस्काउंट, सेल मूल्य, क़ीमत तय करने के लिए मार्कअप और मार्जिन, GST/VAT, प्रतिशत बदलाव, मूल मूल्य खोजने के लिए उल्टा प्रतिशत, प्रतिशत अंक, टिप का बँटवारा, चक्रवृद्धि और कोई संख्या कुल का कितना अंश है।

बिना झंझट तेज़ जवाब। न विज्ञापन, न रजिस्ट्रेशन, न ट्रैकिंग — हिस्ट्री, पिन और सेटिंग्स आपके डिवाइस पर रहती हैं।

## मुख्य विशेषताएँ

### कैलकुलेटर
- **डिस्काउंट** — सेल मूल्य और बचत
- **प्रतिशत बदलाव** — वृद्धि या कमी
- **उल्टा प्रतिशत** — प्रतिशत लागू होने से पहले की क़ीमत
- **मूल मूल्य** — डिस्काउंट से पहले
- **मार्कअप** — लागत + मार्कअप से बिक्री मूल्य
- **मार्जिन / लाभ**
- **GST / टैक्स** — जोड़ें या हटाएँ
- **प्रतिशत अंक**
- **% किसी संख्या का**
- **कितना %**
- **आवश्यक प्रतिशत बदलाव**
- **टिप और बँटवारा**
- **चक्रवृद्धि**
- **कुल का अंश**

### इतिहास और सहूलियत

- **पिन** अक्सर इस्तेमाल वाले हिसाब
- **एक टैप कॉपी**
- **साझा / एक्सपोर्ट** इतिहास
- **फ़ॉर्मैट और राउंडिंग** एडजस्टेबल

### गोपनीयता

- **ट्रैकिंग रहित**

## उपयोग के परिदृश्य

| स्थिति | PercentSnap क्या करता है |
|--------|--------------------------|
| दुकान में सेल | डिस्काउंट → मूल और % off → सेल मूल्य और बचत |
| रेस्तरां में टिप | टिप + बँटवारा → बिल, %, लोग → प्रति व्यक्ति |
| उल्टा GST | GST → कर सहित → कर रहित |
| रिसेल मूल्य | मार्कअप → लागत + मार्कअप → बिक्री मूल्य |
| उत्पाद लाभ | मार्जिन → लागत + बिक्री मूल्य |
| "12% बढ़ने पर कितना" | प्रतिशत बदलाव |
| "₹240 -20% के बाद, मूल?" | उल्टा प्रतिशत |
| साल-दर-साल बदलाव | आवश्यक प्रतिशत बदलाव |
| दीर्घकालीन बचत | चक्रवृद्धि |
| सर्वेक्षण | कुल का अंश |

## यह कैसे काम करता है

**सिस्टम कैलकुलेटर के बजाय अलग ऐप क्यों?**
सिस्टम कैलकुलेटर में "डिस्काउंट", "GST हटाना", "उल्टा प्रतिशत" या "टिप + बँटवारा" के लिए UI नहीं होते। PercentSnap हर फ़ील्ड को लेबल करता है और उपयोग किया गया फ़ॉर्मूला दिखाता है।

**क्या गणना ऑनलाइन होती है?**
नहीं। सब कुछ डिवाइस पर, बिना इंटरनेट।

**क्या मैं अक्सर इस्तेमाल वाले हिसाब सहेज सकता/सकती हूँ?**
हाँ, पिन करें।

**ऑफ़लाइन काम करता है?**
पूरी तरह।
