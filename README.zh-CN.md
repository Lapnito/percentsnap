## PercentSnap 是什么？

**PercentSnap** 是一款专注、快速的百分比计算器，覆盖日常真正会用到的计算：商场折扣、促销价、定价的加成与利润率、增值税、百分比变化、用反向百分比找出原始值、百分点、小费分摊、复利增长，以及数字占总量的比例。

打开应用，点选所需计算器，输入数字，看结果。无广告、无注册、无追踪 — 历史、置顶与设置都留在你的设备上。

## 主要功能

### 计算器
- **折扣** — 折后价与节省金额
- **百分比变化** — 增加或减少
- **反向百分比** — 应用某百分比之前的值
- **原始价格** — 折扣前
- **加成** — 由成本与加成得出售价
- **利润率 / 利润**
- **增值税 / 税金** — 加上或扣除
- **百分点**
- **某数的百分比**
- **是百分之几**
- **所需百分比变化**
- **小费与分摊**
- **复利增长**
- **占总量比例**

### 历史与便捷
- **本地历史**
- **置顶**常用计算
- **一键复制**结果
- **分享 / 导出**历史
- **格式与四舍五入**可调

### 隐私
- **无广告**
- **无追踪**
- **无账号**
- **本机处理**

## 使用场景

| 场景 | 应用作用 |
|------|----------|
| 商场打折 | 折扣 → 原价与折扣 → 折后价与节省 |
| 餐厅小费 | 小费+分摊 → 账单、%、人数 → 每人 |
| 反推增值税 | 增值税 → 含税价 → 不含税 |
| 经销定价 | 加成 → 成本 + 加成 → 售价 |
| 产品利润 | 利润率 → 成本 + 售价 |
| "+12% 是多少" | 百分比变化 |
| "240 元-20% 后，原价多少" | 反向百分比 |
| 年度变化 | 所需百分比变化 |
| 长期储蓄 | 复利增长 |
| 调研占比 | 占总量比例 |

## 工作方式

**为什么用专门应用而不是系统计算器？**
系统计算器没有"折扣"、"减去增值税"、"反向百分比"或"小费+分摊"的专门界面。PercentSnap 为每个输入加注，并显示所用公式。

**计算是否在线？**
不是。全部本机完成，不需要网络。

**能否保存常用计算？**
能，置顶即可。

**能否离线使用？**
完全可以。
