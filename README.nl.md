## Wat is PercentSnap?

**PercentSnap** is een gerichte, snelle procentcalculator voor wat je écht uitrekent in het dagelijks leven: kortingen in de winkel, aanbiedingsprijzen, opslag en marge bij prijzen, btw, procentuele verandering, omgekeerd percentage om een oorspronkelijke waarde te vinden, procentpunten, fooi splitsen, samengestelde groei en welk aandeel een getal vormt van een totaal.

Snelle antwoorden zonder ruis. Geen reclame, geen aanmelding, geen tracking — geschiedenis, vastgezette items en instellingen blijven op je toestel.

## Belangrijkste functies

### Calculators
- **Korting** — aanbiedingsprijs en besparing
- **Procentuele verandering** — toename of afname
- **Omgekeerd percentage** — oorspronkelijke waarde vóór percentage
- **Oorspronkelijke prijs** — vóór een korting
- **Opslag** — verkoopprijs uit kosten + opslag
- **Marge / winst** — marge en winst
- **Btw / belasting** — toevoegen of verwijderen
- **Procentpunten** — verschil in p.p.
- **% van een getal** — hoeveel is X% van Y
- **Welk %** — X is welk % van Y
- **Vereiste verandering** — % nodig van A naar B
- **Fooi & splitsen**
- **Samengestelde groei**
- **Aandeel van totaal**

### Geschiedenis en gemak
- **Lokale geschiedenis**
- **Vastpinnen** van veelgebruikte berekeningen
- **Eén tik kopiëren**
- **Delen / exporteren** geschiedenis
- **Indeling en afronding** instelbaar

### Privacy
- **Geen reclame**
- **Geen tracking**
- **Geen account**
- **Op je toestel**

## Gebruiksscenario's

| Scenario | Wat PercentSnap doet |
|----------|-----------------------|
| Uitverkoop | Korting → originele prijs en % af → prijs en besparing |
| Fooi in restaurant | Fooi & splitsen → rekening, %, personen → per persoon |
| Btw terugrekenen | Btw → bruto → netto |
| Wederverkoper-prijzen | Opslag → kosten + opslag → verkoopprijs |
| Productwinst | Marge → kosten + verkoopprijs → marge en winst |
| "Hoe duur na +12%?" | Procentuele verandering |
| "240 € na -20%, originele prijs?" | Omgekeerd percentage |
| Jaar-op-jaar verandering | Vereiste verandering |
| Lange termijn sparen | Samengestelde groei |
| Onderzoek | Aandeel van totaal |

## Hoe het werkt

**Waarom een aparte app?**
De systeemcalculator heeft geen UI's voor "korting", "btw weghalen", "omgekeerd percentage" of "fooi & splitsen". PercentSnap labelt alle invoer en laat de gebruikte formule zien.

**Rekenen via internet?**
Nee. Alles op het toestel, zonder internet.

**Veelgebruikte berekeningen opslaan?**
Ja, vastpinnen.

**Werkt offline?**
Volledig.
