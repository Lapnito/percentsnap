## Apa itu PercentSnap?

**PercentSnap** adalah kalkulator persentase cepat dan fokus untuk apa yang benar-benar perlu Anda hitung sehari-hari: diskon di toko, harga obral, markup dan margin untuk penetapan harga, PPN, perubahan persentase, persentase terbalik untuk menemukan nilai asli, poin persen, pembagian tip, pertumbuhan majemuk, dan berapa porsi sebuah angka dari total.

Jawaban cepat tanpa kekacauan. Tanpa iklan, tanpa pendaftaran, tanpa pelacakan — riwayat, item disematkan, dan pengaturan tetap di perangkat.

## Fitur utama

### Kalkulator
- **Diskon** — harga setelah diskon dan penghematan
- **Perubahan persentase** — kenaikan atau penurunan
- **Persentase terbalik** — nilai sebelum persentase diterapkan
- **Harga asli** — sebelum diskon
- **Markup** — harga jual dari biaya + markup
- **Margin / laba**
- **PPN / pajak** — tambah atau kurangi
- **Poin persen**
- **% dari sebuah angka**
- **Berapa persen**
- **Perubahan persentase yang dibutuhkan**
- **Tip dan pembagian**
- **Pertumbuhan majemuk**
- **Bagian dari total**

### Riwayat dan kemudahan
- **Riwayat lokal**
- **Sematkan** kalkulasi yang sering dipakai
- **Salin sekali ketuk**
- **Bagikan / ekspor** riwayat
- **Format dan pembulatan** dapat disesuaikan

### Privasi
- **Tanpa iklan**
- **Tanpa pelacakan**
- **Tanpa akun**
- **Di perangkat**

## Skenario penggunaan

| Situasi | Yang dilakukan PercentSnap |
|---------|------------------------------|
| Diskon di toko | Diskon → harga awal dan % → harga akhir dan penghematan |
| Tip restoran | Tip + bagi → tagihan, %, orang → per orang |
| PPN terbalik | PPN → harga termasuk → tanpa PPN |
| Harga jual ulang | Markup → biaya + markup → harga jual |
| Laba produk | Margin → biaya + harga jual |
| "Berapa kalau +12%" | Perubahan persentase |
| "Rp240.000 setelah -20%, asalnya?" | Persentase terbalik |
| Perubahan tahun ke tahun | Perubahan yang dibutuhkan |
| Tabungan jangka panjang | Pertumbuhan majemuk |
| Survei | Bagian dari total |

## Cara kerja

**Mengapa pakai aplikasi khusus, bukan kalkulator sistem?**
Kalkulator sistem tidak punya UI khusus untuk "diskon", "lepaskan PPN", "persentase terbalik", atau "tip + bagi". PercentSnap memberi label setiap kolom dan menunjukkan rumus yang dipakai.

**Apakah perhitungan online?**
Tidak. Semuanya di perangkat, tanpa internet.

**Bisakah saya menyimpan kalkulasi yang sering dipakai?**
Bisa, dengan menyematkannya.

**Apakah berfungsi luring?**
Sepenuhnya.
