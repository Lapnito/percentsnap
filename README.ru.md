## Что такое PercentSnap?

**PercentSnap** — сфокусированный быстрый калькулятор процентов для того, что реально приходится считать в жизни: скидки в магазине, акционные цены, наценка и маржа при ценообразовании, НДС, процентное изменение, обратный процент для нахождения исходного значения, процентные пункты, деление чаевых, сложный рост и какую долю от целого составляет число.

Быстрые ответы без шума. Без рекламы, без регистрации и без слежки — история, закреплённые расчёты и настройки остаются на устройстве.

## Главные возможности

### Калькуляторы
- **Скидка** — акционная цена и экономия
- **Процентное изменение** — рост или снижение
- **Обратный процент** — исходное значение до процента
- **Исходная цена** — до скидки
- **Наценка** — цена продажи из себестоимости и наценки
- **Маржа / прибыль**
- **НДС / налог** — добавить или вычесть
- **Процентные пункты**
- **% от числа**
- **Какой %**
- **Необходимое процентное изменение**
- **Чаевые и деление**
- **Сложный рост**
- **Доля от целого**

### История и удобство
- **Локальная история**
- **Закрепить** часто используемые расчёты
- **Копировать** одним касанием
- **Поделиться / экспортировать** историю
- **Формат и округление** настраиваются

### Конфиденциальность
- **Без рекламы**
- **Без слежки**
- **Без аккаунта**
- **На устройстве**

## Сценарии

| Ситуация | Что делает PercentSnap |
|----------|--------------------------|
| Распродажа | Скидка → цена и % → итог и экономия |
| Чаевые в ресторане | Чаевые + деление → счёт, %, люди → на человека |
| Обратный НДС | НДС → цена с НДС → без НДС |
| Цены для перепродажи | Наценка → себестоимость + наценка → цена продажи |
| Прибыль | Маржа → себестоимость + цена продажи |
| "Сколько +12 %" | Процентное изменение |
| "240 ₽ после -20 %, исходное?" | Обратный процент |
| Год к году | Необходимое процентное изменение |
| Долгосрочные сбережения | Сложный рост |
| Опросы | Доля от целого |

## Как это работает

**Зачем отдельное приложение, а не системный калькулятор?**
Системный не имеет UI для «скидки», «снять НДС», «обратного процента» или «чаевых + деление». PercentSnap подписывает каждое поле и показывает использованную формулу.

**Расчёты онлайн?**
Нет. Всё на устройстве, без интернета.

**Можно сохранить частые расчёты?**
Да, закрепите.

**Работает офлайн?**
Полностью.
