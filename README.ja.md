## PercentSnap とは？

**PercentSnap** は、日常で本当に必要な計算に絞った高速パーセント計算機です。買い物の割引、セール価格、価格設定のためのマークアップと利益率、消費税、変化率、元の値を求める逆算パーセント、ポイント差、チップの割り勘、複利、ある数が全体に占める割合。

雑音のない速い答え。広告なし、登録なし、追跡なし — 履歴・ピン・設定はすべて端末内に。

## 主な機能

### 計算機
- **割引** — セール価格と節約額
- **変化率** — 増減
- **逆算パーセント** — 適用前の値
- **元の価格** — 割引前
- **マークアップ** — 原価とマークアップから販売価格
- **利益率 / 利益**
- **消費税** — 加算・外し
- **ポイント差**
- **数値の％**
- **何％か**
- **必要な変化率**
- **チップ＆割り勘**
- **複利**
- **全体に占める割合**

### 履歴と利便性
- **ローカル履歴**
- **よく使う計算をピン留め**
- **ワンタップでコピー**
- **履歴の共有 / エクスポート**
- **桁数と丸め** 設定可

### プライバシー
- **広告なし**
- **追跡なし**
- **アカウントなし**
- **端末内**

## 利用シーン

| 場面 | アプリの動作 |
|------|--------------|
| セール | 割引 → 元値と割引 → 価格と節約 |
| 飲食店のチップ | チップ＋割り勘 → 会計、％、人数 → 1 人あたり |
| 税抜き計算 | 消費税 → 税込み → 税抜き |
| 仕入れ価格 | マークアップ → 原価＋マークアップ → 販売価格 |
| 商品の利益 | 利益率 → 原価＋販売価格 |
| 「12% 上がるといくら」 | 変化率 |
| 「240 円が −20% 後、元値は」 | 逆算パーセント |
| 前年比 | 必要な変化率 |
| 長期積立 | 複利 |
| アンケート構成比 | 全体に占める割合 |

## 仕組み

**標準電卓ではなく専用アプリを使う理由は？**
標準電卓には「割引」「税抜き」「逆算パーセント」「チップ＋割り勘」専用 UI がありません。PercentSnap は各入力欄に名前を付け、使った式も表示します。

**計算はオンラインで？**
いいえ。すべて端末内、インターネット不要。

**よく使う計算を保存できる？**
はい、ピン留めで。

**オフラインで使える？**
完全に。
