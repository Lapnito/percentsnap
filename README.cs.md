## Co je PercentSnap?

**PercentSnap** je soustředěná, rychlá kalkulačka procent pro to, co se v běžném životě reálně počítá: slevy v obchodě, akční ceny, přirážka a marže pro tvorbu cen, DPH, procentní změna, opačné procento pro nalezení původní hodnoty, procentní body, dělení dýška, složené úročení a jaký podíl na celku tvoří dané číslo.

Rychlé odpovědi bez balastu. Žádné reklamy, žádná registrace, žádné sledování — historie, připnuté výpočty a nastavení zůstávají v zařízení.

## Hlavní funkce

### Kalkulačky
- **Sleva** — akční cena a úspora
- **Procentní změna** — nárůst nebo pokles
- **Opačné procento** — původní hodnota před aplikovaným procentem
- **Původní cena** — před slevou
- **Přirážka** — prodejní cena z nákladu a přirážky
- **Marže / zisk**
- **DPH / daň** — přidat nebo odečíst
- **Procentní body**
- **% z čísla**
- **Kolik %**
- **Potřebná procentní změna**
- **Dýško a rozdělení**
- **Složené úročení**
- **Podíl na celku**

### Historie a pohodlí
- **Lokální historie**
- **Připnutí** často používaných výpočtů
- **Kopírování** výsledku jedním ťuknutím
- **Sdílení / export** historie
- **Formát a zaokrouhlení** nastavitelné

### Soukromí
- **Bez reklam**
- **Bez sledování**
- **Bez účtu**
- **Vše v zařízení**

## Příklady použití

| Situace | Co aplikace udělá |
|---------|-------------------|
| Sleva v obchodě | Sleva → původní cena a % → akční cena a úspora |
| Dýško v restauraci | Dýško + dělení → účet, %, lidé → na osobu |
| Zpětný výpočet DPH | DPH → cena s DPH → bez DPH |
| Ceny pro další prodej | Přirážka → náklad + přirážka → prodejní cena |
| Zisk z produktu | Marže → náklad + prodejní cena → marže a zisk |
| "Kolik je +12 %" | Procentní změna |
| "240 Kč po -20 %, kolik to bylo?" | Opačné procento |
| Meziroční změna | Potřebná procentní změna |
| Dlouhodobé úspory | Složené úročení |
| Rozdělení v průzkumu | Podíl na celku |

## Jak to funguje

**Proč dedikovaná aplikace místo systémové kalkulačky?**
Systémová kalkulačka nemá UI pro "slevu", "DPH zpět", "opačné procento" nebo "dýško + dělení". PercentSnap označí každé pole a ukáže použitý vzorec.

**Probíhají výpočty online?**
Ne. Vše v zařízení, bez internetu.

**Můžu si uložit často používané výpočty?**
Ano, jen je připněte.

**Funguje offline?**
Plně.
