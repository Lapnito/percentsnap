## PercentSnap คืออะไร?

**PercentSnap** คือเครื่องคิดเลขเปอร์เซ็นต์ที่รวดเร็วและตรงจุด สำหรับสิ่งที่คุณต้องคำนวณจริง ๆ ในแต่ละวัน เช่น ส่วนลดในร้าน ราคาขายลด การบวกราคาและกำไรสำหรับตั้งราคา VAT การเปลี่ยนแปลงเปอร์เซ็นต์ เปอร์เซ็นต์ย้อนกลับเพื่อหาค่าตั้งต้น จุดเปอร์เซ็นต์ การหารทิป ดอกเบี้ยทบต้น และเศษส่วนของจำนวนต่อยอดรวม

คำตอบเร็วโดยไม่มีของฟุ่มเฟือย ไม่มีโฆษณา ไม่ต้องลงทะเบียน ไม่มีการติดตาม — ประวัติ การปักหมุด และการตั้งค่าทั้งหมดอยู่ในเครื่องของคุณ

## คุณสมบัติหลัก

### เครื่องคิดเลข
- **ส่วนลด** — ราคาหลังลดและเงินที่ประหยัด
- **เปอร์เซ็นต์การเปลี่ยนแปลง** — เพิ่มหรือลด
- **เปอร์เซ็นต์ย้อนกลับ** — ค่าก่อนใช้เปอร์เซ็นต์
- **ราคาตั้งต้น** — ก่อนลด
- **บวกราคา** — ราคาขายจากต้นทุน + บวกราคา
- **กำไร / มาร์จิ้น**
- **VAT / ภาษี** — บวกหรือหัก
- **จุดเปอร์เซ็นต์**
- **% ของจำนวน**
- **เป็นกี่ %**
- **เปอร์เซ็นต์ที่ต้องเปลี่ยน**
- **ทิปและการหาร**
- **ดอกเบี้ยทบต้น**
- **เศษส่วนของยอดรวม**

### ประวัติและความสะดวก
- **ประวัติในเครื่อง**
- **ปักหมุด** การคำนวณที่ใช้บ่อย
- **คัดลอกผลแตะเดียว**
- **แชร์ / ส่งออก** ประวัติ
- **รูปแบบและการปัด** ปรับได้

### ความเป็นส่วนตัว
- **ไม่มีโฆษณา**
- **ไม่มีการติดตาม**
- **ไม่ต้องมีบัญชี**
- **บนเครื่อง**

## กรณีใช้งาน

| สถานการณ์ | PercentSnap ทำอะไร |
|-----------|---------------------|
| ลดราคาในร้าน | ส่วนลด → ราคาตั้งต้นและ % → ราคาหลังลดและเงินประหยัด |
| ทิปในร้านอาหาร | ทิป + หาร → บิล, %, คน → ต่อคน |
| VAT ย้อนกลับ | VAT → ราคารวม → ราคาก่อน VAT |
| ตั้งราคาขายต่อ | บวกราคา → ต้นทุน + บวกราคา → ราคาขาย |
| กำไรสินค้า | กำไร → ต้นทุน + ราคาขาย |
| "ราคาเพิ่ม 12% เท่าไหร่" | เปอร์เซ็นต์การเปลี่ยนแปลง |
| "240 บาท หลังลด 20% ราคาก่อนหน้า?" | เปอร์เซ็นต์ย้อนกลับ |
| เปลี่ยนแปลงรายปี | เปอร์เซ็นต์ที่ต้องเปลี่ยน |
| ออมระยะยาว | ดอกเบี้ยทบต้น |
| สำรวจความเห็น | เศษส่วนของยอดรวม |

## ทำงานอย่างไร

**ทำไมต้องใช้แอปเฉพาะ ไม่ใช้เครื่องคิดเลขของเครื่อง?**
เครื่องคิดเลขของเครื่องไม่มี UI สำหรับ "ส่วนลด" "หัก VAT" "เปอร์เซ็นต์ย้อนกลับ" หรือ "ทิป + หาร" PercentSnap ใส่ป้ายชื่อทุกช่องกรอก และแสดงสูตรที่ใช้

**คำนวณออนไลน์หรือเปล่า?**
ไม่ ทุกอย่างทำในเครื่อง ไม่ต้องใช้อินเทอร์เน็ต

**บันทึกสูตรที่ใช้บ่อยได้ไหม?**
ได้ ปักหมุดไว้

**ใช้ออฟไลน์ได้ไหม?**
ได้เต็มรูปแบบ
