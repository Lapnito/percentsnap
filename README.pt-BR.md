## O que é o PercentSnap?

**PercentSnap** é uma calculadora de porcentagem rápida e direta, focada nas contas que você realmente faz no dia a dia: descontos na loja, preço promocional, markup e margem para precificar, ICMS/IVA, variação percentual, porcentagem reversa para descobrir o valor original, pontos percentuais, divisão de gorjeta, juros compostos e qual porção do total um número representa.

Tudo é feito no aparelho. Sem anúncios, sem cadastro, sem rastreamento — histórico, fixados e configurações ficam no seu telefone.

## Recursos principais

### Calculadoras
- **Desconto** — preço final e economia a partir do preço cheio
- **Variação percentual** — aumento ou redução entre dois números
- **Porcentagem reversa** — descobre o valor antes da porcentagem
- **Preço original** — quanto custava antes do desconto
- **Markup** — preço de venda a partir do custo + markup
- **Margem / lucro** — margem e lucro do custo + preço de venda
- **ICMS / IVA / imposto** — adicionar ou retirar
- **Pontos percentuais** — diferença em p.p.
- **% de um número** — quanto é X% de Y
- **Qual % é** — X é qual % de Y
- **Variação necessária** — % preciso para sair de A para B
- **Gorjeta e divisão** — gorjeta + divisão da conta
- **Juros compostos** — crescimento composto
- **Porção do total** — parte como % do total

### Histórico e conveniência
- **Histórico local** — toda conta fica salva no aparelho
- **Fixe** as contas mais usadas
- **Copiar** resultado em um toque
- **Compartilhar / exportar** histórico
- **Formato e arredondamento** ajustáveis

### Privacidade
- **Sem anúncios**
- **Sem rastreamento**
- **Sem conta**
- **No aparelho** — histórico, fixados e configurações ficam locais

## Casos de uso

| Cenário | O que o PercentSnap faz |
|---------|--------------------------|
| Liquidação | Desconto → preço cheio + % off → preço final e economia |
| Restaurante | Gorjeta + divisão → conta, % e pessoas → por pessoa |
| ICMS reverso | ICMS/IVA → preço com imposto → preço sem |
| Preços para revenda | Markup → custo + markup → preço |
| Lucro | Margem → custo + preço de venda → margem e lucro |
| "Quanto fica 12% mais caro?" | Variação percentual |
| "Paguei R$240 com -20%, era quanto?" | Porcentagem reversa |
| Variação ano-a-ano | Variação necessária |
| Investimento | Juros compostos |
| Pesquisa de opinião | Porção do total |

## Como funciona

**Por que um app dedicado e não a calculadora do sistema?**
A do sistema não tem UI específica para "desconto", "tirar imposto" ou "porcentagem reversa". O PercentSnap rotula cada campo e mostra a fórmula usada.

**As contas são feitas online?**
Não. Tudo no aparelho. O app não precisa de internet.

**Posso salvar contas frequentes?**
Pode. Fixe a conta e ela aparece no topo.

**Funciona offline?**
Totalmente.
