## ما هو PercentSnap؟

**PercentSnap** حاسبة نسب مئوية سريعة ومركّزة لما تحتاجه فعلًا في الحياة اليومية: خصومات في المتجر، أسعار العروض، الهامش والربح في التسعير، ضريبة القيمة المضافة، تغيّر النسبة، النسبة العكسية لإيجاد القيمة الأصلية، نقاط النسبة، تقسيم البقشيش، النمو المركّب، وما الجزء الذي يمثله رقم من الإجمالي.

إجابات سريعة بلا تشويش. لا إعلانات، لا تسجيل، لا تتبع — السجل والتثبيتات والإعدادات تبقى على جهازك.

## أهم الميزات

### الحاسبات
- **خصم** — السعر بعد الخصم والتوفير
- **تغيّر النسبة** — زيادة أو نقصان
- **نسبة عكسية** — القيمة قبل تطبيق النسبة
- **السعر الأصلي** — قبل الخصم
- **هامش (markup)** — سعر البيع من التكلفة والهامش
- **هامش / ربح**
- **ضريبة / VAT** — أضف أو احذف
- **نقاط النسبة**
- **% من رقم**
- **ما النسبة**
- **النسبة المطلوبة للتغيير**
- **بقشيش وتقسيم**
- **نمو مركب**
- **جزء من الإجمالي**

### السجل والراحة
- **سجل محلي**
- **تثبيت** الحسابات المتكررة
- **نسخ** بنقرة واحدة
- **مشاركة / تصدير** السجل
- **تنسيق وتقريب** قابلان للضبط

### الخصوصية
- **بدون إعلانات**
- **بدون تتبع**
- **بدون حساب**
- **على الجهاز**

## حالات الاستخدام

| الحالة | ماذا يفعل PercentSnap |
|--------|-----------------------|
| تخفيضات | خصم → السعر و% → السعر النهائي والتوفير |
| بقشيش بالمطعم | بقشيش + تقسيم → الفاتورة، %، الأشخاص → للفرد |
| ضريبة عكسية | VAT → السعر شامل → بدون VAT |
| تسعير لإعادة البيع | هامش → التكلفة + الهامش → سعر البيع |
| ربح المنتج | هامش → التكلفة + سعر البيع |
| "كم بزيادة 12%" | تغيّر النسبة |
| "240 ر.س بعد -20%، الأصل؟" | نسبة عكسية |
| تغيّر سنوي | النسبة المطلوبة للتغيير |
| الادخار طويل الأمد | نمو مركب |
| استطلاعات | جزء من الإجمالي |

## كيف يعمل

**لماذا تطبيق مخصص بدلًا من حاسبة النظام؟**
حاسبة النظام لا تملك واجهات لـ "خصم" أو "إزالة VAT" أو "نسبة عكسية" أو "بقشيش وتقسيم". يحدّد PercentSnap كل خانة ويعرض الصيغة المستخدمة.

**هل العمليات على الإنترنت؟**
لا. كلها على الجهاز، بدون إنترنت.

**هل يمكنني حفظ حسابات متكررة؟**
نعم بالتثبيت.

**يعمل دون اتصال؟**
بالكامل.
