## Czym jest PercentSnap?

**PercentSnap** to skoncentrowany, szybki kalkulator procentów do tego, co naprawdę liczy się na co dzień: zniżki w sklepie, ceny promocyjne, narzut i marża przy ustalaniu cen, VAT, zmiana procentowa, procent odwrotny do znalezienia wartości pierwotnej, punkty procentowe, dzielenie napiwku, procent składany i jaki udział w całości stanowi liczba.

Szybkie odpowiedzi bez bałaganu. Bez reklam, rejestracji i śledzenia — historia, przypięte i ustawienia zostają na urządzeniu.

## Najważniejsze funkcje

### Kalkulatory
- **Zniżka** — cena promocyjna i oszczędność
- **Zmiana procentowa** — wzrost lub spadek
- **Procent odwrotny** — wartość przed zastosowaniem procentu
- **Cena pierwotna** — przed zniżką
- **Narzut** — cena sprzedaży z kosztu i narzutu
- **Marża / zysk**
- **VAT / podatek** — dodaj lub odejmij
- **Punkty procentowe**
- **% liczby**
- **Jaki %**
- **Wymagana zmiana procentowa**
- **Napiwek i podział**
- **Procent składany**
- **Udział w całości**

### Historia i wygoda
- **Lokalna historia**
- **Przypnij** często używane obliczenia
- **Kopiuj** wynik jednym dotknięciem
- **Udostępnij / eksportuj** historię
- **Format i zaokrąglanie** regulowane

### Prywatność
- **Bez reklam**
- **Bez śledzenia**
- **Bez konta**
- **Na urządzeniu**

## Zastosowania

| Sytuacja | Co robi PercentSnap |
|----------|---------------------|
| Wyprzedaż | Zniżka → cena pełna i % → cena końcowa i oszczędność |
| Napiwek | Napiwek + podział → rachunek, %, osoby → na osobę |
| Odwrócenie VAT | VAT → cena z VAT → bez VAT |
| Ceny do odsprzedaży | Narzut → koszt + narzut → cena |
| Zysk z produktu | Marża → koszt + cena sprzedaży → marża i zysk |
| "Ile +12%" | Zmiana procentowa |
| "240 zł po -20%, ile było?" | Procent odwrotny |
| Zmiana rok do roku | Wymagana zmiana procentowa |
| Inwestycje | Procent składany |
| Ankiety | Udział w całości |

## Jak to działa

**Po co dedykowana aplikacja zamiast systemowej?**
Systemowy kalkulator nie ma UI dla "zniżki", "VAT-u zdjętego", "procentu odwrotnego" czy "napiwku + podziału". PercentSnap nazywa każde pole i pokazuje użytą formułę.

**Czy obliczenia są online?**
Nie. Wszystko na urządzeniu, bez Internetu.

**Czy mogę zapisać częste obliczenia?**
Tak, przypinając.

**Działa offline?**
W całości.
